## Solución para filesystem-ultra
Problema: wsl_to_windows_copy solo busca en /home/claude/, no en /mnt/c/
Opciones:
1. Modificar el MCP (recomendado)
Edita el código fuente de filesystem-ultra para que wsl_to_windows_copy acepte rutas /mnt/c/:
rust// En el handler de wsl_to_windows_copy
let wsl_path = if wsl_path.starts_with("/mnt/") {
    wsl_path.to_string()  // Ya es ruta WSL válida
} else {
    format!("/home/claude/{}", wsl_path)  // Añadir prefijo solo si necesario
}
2. Usar auto-sync (más fácil)
Activa el auto-sync para que copie automáticamente:
bash# Dentro del MCP, configura:
configure_autosync(enabled: true, sync_on_write: true)
Esto sincronizará automáticamente cada vez que escribas en /mnt/c/ a su equivalente Windows.
3. Trabajar solo en Windows
Si filesystem-ultra ya edita correctamente en C:\__REPOS\... (como está haciendo ahora), no necesitas wsl_to_windows_copy. Los cambios ya están en Windows.
