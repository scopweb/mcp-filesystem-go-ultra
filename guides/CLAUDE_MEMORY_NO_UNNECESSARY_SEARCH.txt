# Claude Desktop - Prevención de Búsquedas Innecesarias

## Regla de Oro

**ANTES de buscar (smart_search/mcp_search), PREGUNTA al usuario:**

```
❌ "Déjame buscar dónde está X..."
✅ "¿Sabes en qué archivo/línea está X? Si no, lo busco."
```

## Solo Busca Si:

1. Usuario dice "no sé dónde está"
2. Usuario pide explícitamente "busca X" o "encuentra X"
3. Usuario claramente no tiene la info

## Ejemplos:

**Caso 1: Usuario conoce archivo**
```
Usuario: "Modifica ProcessData en main.go"
Tú: "¿Sabes en qué línea está? Si no, la busco."
Usuario: "Línea 150"
Tú: [Lee línea 150, edita] ✅ Ahorro 90%
```

**Caso 2: Usuario no sabe**
```
Usuario: "Busca todas las funciones con 'timeout'"
Tú: [Ejecuta smart_search] ✅ Búsqueda necesaria
```

**Caso 3: Usuario da ubicación completa**
```
Usuario: "En core/engine.go línea 245, cambia timeout a 60"
Tú: [Lee línea 245, edita directamente] ✅ Sin búsqueda
```

## Ahorro de Tokens:

- Con pregunta previa: ~200 tokens
- Con búsqueda automática: ~2,000 tokens
- **Ahorro: 90%** cuando usuario ya sabe ubicación

## Recuerda:

- Preguntar primero = Más rápido + Menos tokens
- Solo buscar cuando sea realmente necesario
- Usar contexto de conversaciones previas

---

**Aplica esta regla SIEMPRE antes de ejecutar operaciones de búsqueda.**
