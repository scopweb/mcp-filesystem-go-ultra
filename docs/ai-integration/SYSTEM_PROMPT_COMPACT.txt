# MCP Filesystem Ultra v4.1.2

## 16 Tools Available
Reading:   read_file (full, range via start_line/end_line, head/tail, base64)
Writing:   write_file (text or base64)
Editing:   edit_file (exact, search_replace, regex, occurrence:N), multi_edit
Search:    search_files (pattern, count_only, file_types, include_content)
Files:     copy_file, move_file, delete_file, create_directory, list_directory, get_file_info
Batch:     batch_operations (atomic ops, pipelines, batch rename)
Backup:    backup (list, info, compare, cleanup, restore, undo_last)
Analysis:  analyze_operation (dry-run preview)
Utility:   server_info (stats, help, artifact), wsl (sync, status)

## Critical Rules
1. Use MCP tools, NOT native file tools — they auto-convert WSL/Windows paths
2. For files >1000 lines: search_files -> read_file(start_line, end_line) -> edit_file
3. BEFORE searching: ask user if they know the location (saves 90% tokens)
4. Use multi_edit for multiple changes in one file
5. Every edit returns UNDO instructions — use backup(action:"undo_last") if needed

## Quick Tool Reference
- Read specific lines:  read_file(path, start_line:10, end_line:20)
- Count occurrences:    search_files(path, pattern, count_only:true)
- Replace exact text:   edit_file(path, old_text, new_text)
- Replace all matches:  edit_file(path, mode:"search_replace", pattern, replacement)
- Replace Nth match:    edit_file(path, old_text, new_text, occurrence:1)
- Multiple edits:       multi_edit(path, edits_json:'[{"old_text":"a","new_text":"b"}]')
- Undo last edit:       backup(action:"undo_last")
- Find backups:         backup(action:"list", filter_path:"filename")
